2. List of objects that can be used in JavaScript

There are four types of objects that can be used in JavaScript

  1. Intrinsic objects
  2. Objects that are created
  3. Host objects
  4. ActiveX objects

Intrinsic objects

array and string are examples of intrinsic objects in JavaScript.

An example of an intrinsic array object:

var myPets = ["Atlas", "Gizmo"];

Objects that are created

For example:

const petObj = new Object(); 
petObj.name="Atlas";
document.write(petObj.name);
// Output: Atlas

Host objects

window and document are two types of host objects.

An example of the window host object is:

const w = window.innerWidth
|| document.documentElement.clientWidth

ActiveX objects

Active X is a Microsoft extension and is for IE only. ActiveX has security flaws so use with caution.

An example of creating an ActiveX Automation Object:

const ExcelApp = new ActiveXObject("Excel.Application");
const ExcelSheet = new ActiveXObject("Excel.Sheet");
